Cara membuat bot Telegram (2 menit, gratis)
  1. Di Telegram, chat ke @BotFather → kirim /newbot → ikuti instruksi → salin token (format 123456:AA...).
  2. Chat ke bot baru Anda sekali (pesan apa saja), lalu buka @userinfobot untuk mengetahui Chat ID Anda.
  3. Masukkan keduanya di atas → Simpan → kirim pesan uji. Setiap aksi autopilot akan dikabari ke chat ini.
Penjadwal di PC Windows 24 jam
  • Default: python run.py menyalakan server web dan scheduler thread otomatis — biarkan window ini terbuka / jalankan sebagai service.
  • Alternatif Windows Task Scheduler: jadwalkan python scripts\run_engine_once.py tiap N menit (lihat README bagian "Penjadwalan").
  • Database: file SQLite autopilot.db di folder aplikasi — cukup copy untuk backup.
