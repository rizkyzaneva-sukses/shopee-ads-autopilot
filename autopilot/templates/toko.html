Untuk mode DEMO Anda boleh mengosongkan Shop ID — sistem membuat id internal. Untuk mode LIVE, toko harus diotorisasi lewat OAuth (panduan di kanan).

🔐 Menghubungkan toko nyata (OAuth resmi Shopee)
  1. Isi SHOPEE_PARTNER_ID & SHOPEE_PARTNER_KEY di file .env (lihat .env.example), restart server.
  2. Jalankan python scripts/authorize_shop.py --nama "Cabang Bandung" → buka URL yang tercetak, login sebagai pemilik toko, klik Setujui.
  3. Salin URL hasil redirect → python scripts/authorize_shop.py --nama "Cabang Bandung" --pasted-url "URL_HASIL".
  4. Ulangi untuk setiap cabang (>5 toko didukung). Token di-refresh otomatis (berlaku ±4 jam).
  5. Scope Ads (modul 117) perlu diaktifkan oleh Shopee Partner Support untuk akun developer Anda.
🛡️ Plafon harian adalah rem terakhir: total budget kampanye ongoing sebuah toko tidak akan pernah dinaikkan autopilot melebihi plafon ini — walau aturan menyuruh naik. Cocok untuk mengontrol total bon iklan per cabang.
