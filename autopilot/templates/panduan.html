1. Apa itu Shopee Ads Autopilot?

▾

Shopee Ads Autopilot adalah aplikasi internal untuk mengotomatiskan pengelolaan iklan Shopee di banyak toko/cabang. Aplikasi ini berjalan di PC Windows Anda sendiri — bukan layanan cloud, bukan SaaS berlangganan.

Untuk siapa aplikasi ini?

  • Pemilik toko Shopee multi-cabang yang ingin iklannya dikelola otomatis (naikkan budget kalau untung, turunkan/pause kalau boncos).
  • Manajer iklan yang perlu melihat performa semua cabang dalam satu dashboard.
  • Siapa saja yang ingin belajar mengelola iklan Shopee dengan aman — karena mode pertama kali adalah DEMO (simulasi, saldo iklan 100% aman).

2. Instalasi & Quick Start (Mode DEMO)

▾

Anda hanya butuh Python 3.10+ yang terinstall di Windows. Tidak perlu Docker, Node.js, atau tools tambahan.

Langkah 1 — Install Python

Jika belum punya Python, download dari python.org dan centang "Add Python to PATH" saat instalasi.

Langkah 2 — Install dependensi

Buka PowerShell atau Command Prompt, lalu jalankan:

cd folder-aplikasi
python -m pip install -r requirements.txt

Langkah 3 — Isi data demo

Perintah ini membuat 6 toko cabang, 30 kampanye, dan 30 hari data performa simulasi:

python scripts\seed_demo.py

Langkah 4 — Jalankan server

python run.py

Langkah 5 — Buka dashboard

Buka browser: http://127.0.0.1:8765

Untuk menjalankan ulang demo dari nol:
python scripts\seed_demo.py --reset

5. Panduan Mengisi Prioritas

▾

Prioritas menentukan urutan evaluasi aturan. Angka kecil = dievaluasi duluan.

Jenis aturanIsi prioritasContoh Anda
⏸ Pause kampanye (paling tegas)5Pause total boncos → 5 ✓
📉 Turunkan budget / rem boncos10Rem kampanye boncos → 10 ✓
📈 Naikkan budget / gas juara20–30Gas kampanye juara → 20 ✓
🔔 Cuma notifikasi / pengingat90Pengingat migrasi → 90 ✓
❓ Tidak tahu / aturan umum50default
Kenapa aturan "rem/pengaman" harus berangka lebih kecil dari aturan "gas"?

Karena aturan dievaluasi sesuai urutan prioritas — aturan dengan angka kecil dievaluasi duluan. Jika sebuah kampanye memenuhi syarat untuk pause (angka prioritas 5) dan sekaligus syarat untuk naikkan budget (angka prioritas 20), maka aturan pause akan dieksekusi terlebih dahulu. Ini memastikan bahwa rem darurat selalu lebih cepat dari akselerasi — budget tidak akan dinaikkan untuk kampanye yang seharusnya dipause. Kebalikannya berbahaya: jika aturan "gas" (angka 20) dievaluasi duluan, budget bisa dinaikkan sejenak sebelum pause mengejar.

💡 Tips: Mulai dengan angka "longgar" (misal pause=5, rem=10, gas=20, notif=90). Sesuaikan nanti berdasarkan hasil di Log.

7. Setup Telegram & Kirim Pesan Uji

▾

Langkah 1 — Buat Bot Telegram

  1. Buka Telegram, cari chat @BotFather
  2. Kirim /newbot
  3. Ikuti instruksi: beri nama bot (misal: "Ads Autopilot") dan username
  4. Salin token yang diberikan (format: 123456:AA...)

Langkah 2 — Dapatkan Chat ID

  1. Kirim pesan apa saja ke bot baru Anda
  2. Buka Telegram, cari chat @userinfobot
  3. Kirim /start — bot akan menampilkan Chat ID Anda

Langkah 3 — Isi di Settings

  1. Buka halaman Settings di dashboard
  2. Isi Telegram Bot Token dengan token dari @BotFather
  3. Isi Telegram Chat ID dengan angka dari @userinfobot
  4. Klik 💾 Simpan settings

Langkah 4 — Kirim Pesan Uji

Klik tombol ✈️ Kirim pesan uji Telegram. Pesan berhasil: ✅ Uji Telegram dari Ads Autopilot berhasil!

8. Menjalankan 24 Jam di PC Windows

▾

Cara Utama: Server + Scheduler Bawaan

python run.py

Scheduler thread bawaan menjalankan siklus evaluasi otomatis tiap interval (default 60 menit, ubah di Settings).

Alternatif: Windows Task Scheduler

schtasks /create /tn "ShopeeAdsAutopilot" ^
  /tr "\"C:\path\ke\python.exe\" \"C:\path\ke\shopee-ads-autopilot\scripts\run_engine_once.py\"" ^
  /sc minute /mo 60

Autostart saat Windows Nyala

  • Cara 1: Buat shortcut run.py → taruh di folder shell:startup
  • Cara 2: Gunakan Task Scheduler dengan trigger At system startup

Backup

Cukup salin file autopilot.db. Jika mode LIVE, juga backup folder tokens/.

9. Roadmap dari DEMO ke LIVE

▾
🗺️ Roadmap Anda setelah ini

Minggu ini (Mode DEMO — tanpa risiko):

  1. Jalankan 3 perintah instalasi → dashboard terbuka dengan 6 toko + 30 kampanye contoh
  2. Coba buat aturan sendiri di halaman Aturan (mis. "ROAS < 2 → turunkan budget 20%")
  3. Klik ▶ Jalankan Siklus Evaluasi di Monitoring → lihat hasilnya di Log Keputusan
  4. Isi Telegram di Settings (@BotFather → token, @userinfobot → chat ID) → "Kirim pesan uji"
  5. Coba pencet ⛔ KILL-SWITCH di pojok kanan atas — rasakan rem daruratnya

Nanti (menuju data toko asli):

  1. Daftarkan app di Shopee Open Platform → dapat partner_id + partner_key → isi ke .env
  2. Email/chat Shopee Partner Support minta scope Ads (modul 117) diaktifkan
  3. Otorisasi tiap cabang: python scripts\authorize_shop.py --nama "Cabang Bandung"
  4. Verifikasi field Tingkat 2 — bilang ke asisten AI saat sampai tahap ini
  5. Baru aktifkan mode LIVE (aturan baru tetap otomatis dry-run 24 jam dulu)

  • DRY-RUN 24 jam per aturan baru
  • Kill-switch, plafon per toko, lantai/plafon per kampanye, maks aksi/hari

11. Troubleshooting & FAQ

▾

Q1: Server tidak mau nyala?

  1. Pastikan Python terinstall: python --version
  2. Install dependensi: python -m pip install -r requirements.txt
  3. Pastikan di folder yang benar sebelum python run.py
  4. Jika ModuleNotFoundError, jalankan ulang pip install

Q2: Port 8765 sudah dipakai?

python run.py --port 9000

Q3: Kampanye tidak muncul?

  1. Jalankan python scripts\seed_demo.py
  2. Jika LIVE: pastikan toko sudah diotorisasi
  3. Pastikan scope Ads (modul 117) aktif
  4. Klik ↻ Sinkron + Evaluasi

Q4: Telegram tidak terkirim?

  1. Pastikan bot sudah pernah dikirim pesan minimal sekali
  2. Cek token dan Chat ID di Settings
  3. Klik ✈️ Kirim pesan uji

Q5: Throttle terus?

Naikkan max_actions_day di form edit aturan, atau tunggu hari berikutnya.

Q6: Reset demo dari nol?

python scripts\seed_demo.py --reset

Q7: Backup?

Salin autopilot.db. Jika LIVE, juga backup folder tokens/.

Q8: Matikan autopilot untuk 1 toko?

Halaman Toko → uncentang checkbox autopilot → 💾.

Q9: Akhiri dry-run lebih awal?

Halaman Aturan → ubah → centang "Akhiri dry-run lebih awal" → Simpan.

Q10: Apakah aman untuk saldo?

  1. Mode awal DEMO — tidak ada API tulis
  2. Dry-run 24 jam per aturan baru
  3. Kill-switch global
  4. Autopilot per toko
  5. Plafon harian per toko
  6. Lantai & plafon per kampanye
  7. Maks aksi/hari
  8. Minta konfirmasi
  9. Log Keputusan permanen
